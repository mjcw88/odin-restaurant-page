# The Odin Project (Full Stack Javascript) - Restaurant Page

Restaurant Page project for the Odin Project Full Stack Javascript course. Primarily focusing on integrating Webpack, and dynamically loading html elements with javascript import/exports.

Full project specs can be found [here](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page).
